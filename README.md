# sms

## 使用说明
```
1.如果没有node.js，请先安装node.js
2.node.js安装完成后，使用vs code或其他编辑器打开项目
3.在项目中打开终端，运行命令-----npm i
4.待所有依赖安装完毕之后，运行命令-----npm run serve
5.打开local网址即可使用(http://localhost:8080/) 端口号存在占用的情况，请以终端运行结果为准
```
